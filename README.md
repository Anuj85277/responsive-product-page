# responsive-product-page

A fully responsive product landing page built using **HTML5, CSS3, and Vanilla JavaScript**, following a Figma design specification.

This project was developed as part of a frontend assignment and focuses on pixel-accurate implementation, performance optimization, and clean code practices.


How to Run the Project
To run this project locally, follow the steps below:
Clone the repository
Bash
Copy code
git clone https://github.com/Anuj85277/responsive-product-page.git

Navigate to the project folder


Open the project

Open the index.html file in any modern web browser.

You can do this by:

Double clicking the index.html file, or

Right clicking the file and selecting Open With → Chrome / Edge / Firefox


Since this project is built using vanilla HTML, CSS, and JavaScript, no additional dependencies, build tools, or installations are required.
